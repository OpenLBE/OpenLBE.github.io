:::summary
How to deploy with SkyRoam.
:::

## Connect
Use the Type-C cable that comes with the device to connect the test host and the device. After a successful connection, the connected device will be displayed. 

<p align="center">
  <img src="./images/unity/deploy.png" alt="deploy" width="60%"/>
</p>

## Register
Select the device to operate on, for example, the data entry A9210, click the Register button to register the device, and a log prompt will appear upon success. One device only needs to be registered once. 

<p align="center">
  <img src="./images/unity/deploy-2.png" alt="deploy-2" width="60%"/>
</p>

## Deploy
Select the device to operate on, for example, the data entry A9210, click the Deploy button to deploy the device, and a log prompt will appear upon success. Perform deployment updates when testing version updates.

  <img src="./images/unity/deploy-3.png" alt="deploy-3" width="60%"/>
</p>
